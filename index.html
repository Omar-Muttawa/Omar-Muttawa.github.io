Lifeline

LifeLine is a low-cost Internet-of-Things vital sign monitor with real-time illness prediction which allows doctors to monitor several patients at once from a remote location.

Learn More

Asthmaway

AsthmAway is a smart inhaler spacer that teaches children how to effectively use their inhaler. With an intuitive hardware design and a beautiful app interface AsthmAway is great at teaching children how to effectively use their inhaler.-

Learn More

Braillio

An electronic kindle-like device which has the fundamental aim of teaching children the Braille alphabet in a fun, interactive and independent way. Through touch and hearing alone, the tablet will enable users to learn to read different letters and even words.

Learn More

Deep Learning Project

A report evaluating the use of different Deep Neural Network architectures in Image representation. This is done by evaluating different model architectures ability to describe noisy patches from the NH-Patches data-set into 128x1 description vectors which are used to represent the patch and differentiate itself from other patches.
